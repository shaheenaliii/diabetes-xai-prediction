
# 🤖 Explainable AI for Diabetes Prediction

## 📌 Overview
This project predicts diabetes risk using machine learning models and explains predictions using Explainable AI techniques.

---

## 🚀 Features
- Logistic Regression, Random Forest, XGBoost  
- Model comparison and evaluation  
- SHAP (global & local explanations)  
- LIME for instance-level explanation  
- What-if simulation and counterfactual analysis  
- Streamlit-based interactive application  

---

## 🛠️ Tech Stack
Python | Streamlit | Scikit-learn | XGBoost | SHAP | LIME  

---
